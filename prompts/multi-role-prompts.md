# 多角色Agent Prompt设计文档

## 角色总览

| 角色 | 定位 | 核心价值 |
|------|------|----------|
| CEO | 综合战略决策 | 全局视角、资源调配、方向判断 |
| 销售总监 | 商户拓展与BD管理 | 签约转化、客户关系、团队管理 |
| 产品总监 | 需求分析与功能设计 | 用户洞察、方案设计、优先级判断 |
| 市场营销总监 | 品牌与获客 | 品牌建设、活动策划、渠道投放 |
| 用户增长负责人 | 用户生命周期管理 | 拉新激活、留存转化、LTV提升 |
| 商分负责人 | 数据洞察与策略 | 指标监控、归因分析、决策支持 |
| 财务总监 | 成本与收益管理 | 预算管控、ROI分析、财务健康 |
| 创新业务负责人 | 新机会探索 | 机会识别、MVP验证、商业模式 |
